\section{Type III Factors and the Split Property}
\label{sec:type_III}

\subsection{Introduction}

Local algebras in quantum field theory are typically type III$_1$ factors. This section discusses the expected structure for CTUFT.

\subsection{Type Classification}

\begin{definition}[Factor Types]
A von Neumann algebra $\mathcal{M}$ with trivial center ($\mathcal{M} \cap \mathcal{M}' = \mathbb{C}\mathbb{I}$) is:
\begin{itemize}
    \item \textbf{Type I}: Has minimal projections (matrix algebras)
    \item \textbf{Type II}: Has finite traces but no minimal projections
    \item \textbf{Type III}: No trace exists
\end{itemize}
\end{definition}

\subsection{CTUFT Local Algebras}

\begin{conjecture}[Type III$_1$ Property]
For any open bounded region $\mathcal{O} \subset \mathbb{R}^4$, the local algebra $\mathcal{A}(\mathcal{O})$ in CTUFT is a type III$_1$ factor.
\end{conjecture}

\textbf{Evidence:}
\begin{enumerate}
    \item Generic QFTs with mass gap have type III local algebras
    \item The torsion field has continuous spectrum (no minimal projections)
    \item Modular theory shows non-trivial $\Delta$ spectrum
\end{enumerate}

\subsection{Split Property}

\begin{conjecture}[Split Property]
For $\mathcal{O}_1 \subset \mathcal{O}_2$ with finite separation, there exists a type I factor $\mathcal{N}$:
\begin{equation}
    \mathcal{A}(\mathcal{O}_1) \subset \mathcal{N} \subset \mathcal{A}(\mathcal{O}_2)
\end{equation}
\end{conjecture}

This property ensures sufficient local degrees of freedom for entanglement.

\subsection{Current Status}

Type III structure and split property are established for free fields. For the interacting CTUFT:
\begin{itemize}
    \item Free limit: Proven type III$_1$
    \item Perturbative: Evidence supports type III
    \item Non-perturbative: Open problem
\end{itemize}
