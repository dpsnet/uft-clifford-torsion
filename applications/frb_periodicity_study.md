# CTUFT-FRB周期性理论研究

## C1. FRB能量-τ₀关系

### 基础关系

FRB爆发能量：
```
E_burst = E_magnetic × (1/τ₀) × η_duty
        = (B²/8π) × V × 200 × η_duty
```

对于典型磁星参数：
- B ~ 10¹⁴-10¹⁵ G
- R ~ 10 km
- η_duty ~ 0.1-0.5（窗口效率）

```
E_burst ~ 10³⁹-10⁴¹ erg × (B/10¹⁴G)²
```

### 能量-周期关联

CTUFT预测能量与周期相关：
```
E_burst ∝ P_spin^(-2) ∝ P_FRB^(-2)
```

因为 E_magnetic ∝ B² ∝ P_spin^(-2)（磁星制动）。

## C2. FRB 121102周期预测

### 观测特征
- 最早发现的重复FRB（2012年）
- 极高重复率，但**无明确周期**
- 活动窗口：可能很窄或很宽

### CTUFT解释

**情形A：短周期（4-8天）**
```
P_spin ~ 0.5-1 小时 → P_FRB ~ 4-8 天
```
问题：受银河系电子密度闪烁影响，短周期难以识别

**情形B：长周期（>100天）**
```
P_spin ~ 12 小时 → P_FRB ~ 100 天
```
问题：超出CHIME等望远镜的连续观测窗口

**情形C：非周期性**
```
如果τ₀-modulated窗口 duty_cycle < 10%
则周期难以从统计中识别
```

## C4. FRB 180916详细验证

### 观测参数
- 周期：16.35 ± 0.15 天
- 活动窗口：~4天（~25% duty cycle）
- 距离：~149 Mpc
- 宿主：螺旋星系恒星形成区

### CTUFT拟合

```
P_FRB = 16.35 天
→ P_spin = 16.35/200 = 0.08175 天 = 1.96 小时
```

这与磁星典型周期范围一致（1-12小时）。

### 活动窗口解释

```
τ₀-modulated window:
- 当τ₀"阀门"打开时：爆发允许（~25%时间）
- 当τ₀"阀门"关闭时：爆发抑制

几何解释：磁轴-自转轴倾角 + τ₀调制
```

### 预测检验

**短期（1-3年）**：
- 精确周期稳定性（τ₀常数预言无漂移）
- 子周期结构（可能的τ₀²调制）

**中期（3-10年）**：
- P_spin spin-down（与年龄关联）
- P_FRB相应增加（验证P_FRB ∝ P_spin）
