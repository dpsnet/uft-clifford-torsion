## B1. 中子星质量-半径关系

### CTUFT修正

标准TOV方程：
```
dP/dr = -G(ε+P)(m+4πr³P)/(r(r-2Gm))
```

CTUFT修正（谱维流效应）：
```
dP/dr = -G(ε+P)(m+4πr³P)/(r(r-2Gm)) × (1 + α·f_in(r))
```

其中 f_in(r) 在中子星内部可达 0.3-0.5。

## B2. 原初黑洞霍金辐射修正

### 标准霍金辐射
```
T_H = ℏc³/(8πGMk_B)
```

### CTUFT修正（谱维流效应）

对于小质量原初黑洞（M < 10¹² kg），内部f_in接近1：
```
T_CTUFT = T_H × (1 + β·(M_Pl/M)² × τ₀)
```

其中 β ~ 0.1。

### 关键效应

1. **辐射谱修正**：
   - 高能尾巴增强（d_s < 4效应）
   - 有效维度流影响态密度

2. **蒸发终点**：
   - 标准：完全蒸发
   - CTUFT：M_min ~ M_Pl × √τ₀ ~ 10⁻³⁵ kg（稳定残余）

### 宇宙学意义
- 暗物质候选者（10⁻¹⁶-10⁻¹² M☉原初黑洞）
- 宇宙射线各向异性

## B3. 宇宙射线能谱（Knee/Ankle解释）

### 标准模型困难
- Knee (~3 PeV)：来源转变？
- Ankle (~3 EeV)：银河系-河外转变？
- GZK截断 (~50 EeV)：理论清晰但观测有超出

### CTUFT解释：τ₀能垒

粒子能量超过临界值时，在4D-10D界面产生τ₀限制：
```
E_c = τ₀⁻¹ × (ZeB·L) ≈ 200 × (ZeB·L)
```

这产生多个特征能量：

| 特征 | 能量 | CTUFT解释 |
|------|------|----------|
| **Knee** | ~3 PeV | 银河系内源τ₀第一次倍增 (E × τ₀) |
| **Second Knee** | ~100 PeV | 重核τ₀效应 |
| **Ankle** | ~3 EeV | 河外源τ₀效应 |
| **GZK修正** | ~50 EeV | 修正的相互作用截面 |

### 预测
- Knee处成分转变：轻核→重核（τ₀对不同电荷依赖）
- 各向异性幅度能量依赖
