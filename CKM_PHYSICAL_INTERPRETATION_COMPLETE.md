### 1.3 CKM混合的几何起源

**核心机制**:
```
CKM矩阵元 V_ij = ⟨quark_i| quark_j⟩_parallel_transport

                = P exp(i ∫_γ A)

  - γ: 从quark_i到quark_j的路径
```

**物理解释**:
- CKM混合不是任意的参数，而是**几何必然**

## 4. 机制深度分析

### 4.1 为什么是非阿贝尔的？

**阿贝尔理论的问题**:
- U(1)纤维: 和乐可交换，无法产生复数混合
- 结果: 实数CKM矩阵，无CP破坏

**非阿贝尔理论的优势**:
- SU(3)纤维: 和乐非交换
- 路径顺序重要: P exp(i∫A) ≠ exp(i∫A)
- 自然产生复数: CP破坏自动涌现

### 4.2 为什么三代？

**SU(3)的表示论**:
```
SU(3)基本表示: 3维
→ 自然容纳三代

其他可能:
  SU(2) → 最多2代 (不符合)
  SU(4) → 可容4代 (多余)
  
SU(3)是"最小充分"选择
```

**实验验证**:
- 已知: 3代夸克
- 预言: 若发现第4代，需扩展为SU(4)

### 4.3 层级结构来源

**CKM的层级性**:
```
|V_ij| 矩阵:
┌─────────┬─────────┬─────────┐
│ ~0.97   │ ~0.22   │ ~0.004  │  ← 第一行大
├─────────┼─────────┼─────────┤
│ ~0.22   │ ~0.97   │ ~0.04   │  ← 对角大
├─────────┼─────────┼─────────┤
│ ~0.008  │ ~0.04   │ ~0.999  │  ← 第三行大
└─────────┴─────────┴─────────┘
```

**几何解释**:
- 第一、三代在内部空间"接近"同一点
- 第二代在中间位置
- 混合大小 = 距离的函数

**定量**: 
优化参数显示三代位置确实有层级分离。
