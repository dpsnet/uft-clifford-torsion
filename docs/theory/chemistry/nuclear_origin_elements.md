# 化学元素深度研究：元素的核起源与宇宙学丰度

## 1. 引言：元素从何而来

### 1.1 核合成概览

宇宙中所有重子物质由以下过程产生：

| 过程 | 时间/地点 | 产生元素 |
|-----|----------|---------|
| 大爆炸核合成 | 宇宙开始后3分钟 | H, He, Li, Be |
| 恒星核合成 | 恒星内部 | C 到 Fe |
| 超新星爆发 | 大质量恒星死亡 | Fe 到 Zr |
| r-过程 | 中子星并合/超新星 | 重元素（A > 100）|
| s-过程 | 渐近巨星分支星 | 重元素（慢中子捕获）|
| p-过程 | 超新星爆炸 | 富质子同位素 |

### 1.2 宇宙元素丰度

**太阳系丰度（按质量）**：
- H: 71%
- He: 27%
- O: 1.0%
- C: 0.4%
- Ne: 0.2%
- Fe: 0.1%
- 其他：~0.3%

**对数丰度曲线**：
```
log(N_X/N_H) vs A（质量数）
```
呈现特征的双峰结构，反映不同核合成机制。

## 2. 大爆炸核合成（BBN）

### 2.1 物理条件

**时间演化**：
- t = 0.01秒：T = 10¹¹ K，kT = 10 MeV
- t = 1秒：T = 10¹⁰ K，kT = 1 MeV
- t = 100秒：T = 10⁹ K，kT = 0.1 MeV（核合成开始）
- t = 1000秒：T = 3×10⁸ K（核合成结束）

**关键反应**：

1. **中子-质子平衡**：
```
n ⇌ p + e⁻ + ν̄_e
```

平衡比：
```
n_n/n_p = exp(-Δmc²/kT) ≈ 1/7 在 T = 0.8 MeV
```

2. **氘合成 bottleneck**：
```
p + n → D + γ
```

温度降至 kT ≈ 0.1 MeV 时才有效（光子不再光解离D）

### 2.2 BBN计算

**核反应网络**：
```
(dn_i/dt) = Σ_j N_j λ_{ji} - n_i Σ_k λ_{ik}
```

主要反应链：
```
p + n → D + γ
D + p → ³He + γ
D + D → ³He + n 或 T + p
T + D → ⁴He + n
³He + D → ⁴He + p
```

### 2.3 BBN预言与观测

**理论预言（标准ΛCDM）**：
```
Y_p (⁴He质量分数) = 0.247 ± 0.001
D/H = (2.6 ± 0.1) × 10⁻⁵
³He/H = (1.0 ± 0.1) × 10⁻⁵
⁷Li/H = (5.0 ± 0.5) × 10⁻¹⁰
```

**观测值**：
- ⁴He：0.244 ± 0.001（吻合）
- D：~(2.5-2.8) × 10⁻⁵（吻合）
- ³He：~(1.1-1.2) × 10⁻⁵（吻合）
- ⁷Li：~1.5 × 10⁻¹⁰（**差异3倍！**——锂缺失问题）

### 2.4 锂缺失问题

**可能解释**：
1. 早期恒星消耗了原初锂
2. BBN期间的非标准物理
3. 系统误差

当前热点研究领域。

## 3. 恒星核合成

### 3.1 氢燃烧（主序星）

**pp链（小质量恒星）**：
```
p + p → D + e⁺ + ν_e      (τ_{1/2} ~ 10⁹ 年)
D + p → ³He + γ
³He + ³He → ⁴He + 2p      (主要终结)
或
³He + ⁴He → ⁷Be + γ
⁷Be + e⁻ → ⁷Li + ν_e
⁷Li + p → 2⁴He
```

**CNO循环（大质量恒星）**：
```
¹²C + p → ¹³N + γ
¹³N → ¹³C + e⁺ + ν_e
¹³C + p → ¹⁴N + γ
¹⁴N + p → ¹⁵O + γ
¹⁵O → ¹⁵N + e⁺ + ν_e
¹⁵N + p → ¹²C + ⁴He  (循环)
或
¹⁵N + p → ¹⁶O + γ  (分支)
```

**能量产生**：
```
4p → ⁴He + 2e⁺ + 2ν_e + 26.7 MeV
```

### 3.2 氦燃烧（红巨星）

**三α过程**：
```
2⁴He ⇌ ⁸Be  (不稳定，τ ~ 10⁻¹⁶ s)
⁸Be + ⁴He → ¹²C* → ¹²C + γ
```

**关键共振**：
Hoyle态（¹²C*, 7.65 MeV）的存在使反应率足够高。

**后续反应**：
```
¹²C + ⁴He → ¹⁶O + γ
¹⁶O + ⁴He → ²⁰Ne + γ
```

### 3.3 碳/氧燃烧

**碳燃烧（T > 5×10⁸ K）**：
```
¹²C + ¹²C → ²⁰Ne + ⁴He
           → ²³Na + p
           → ²³Mg + n
```

**氧燃烧（T > 10⁹ K）**：
```
¹⁶O + ¹⁶O → ²⁸Si + ⁴He
           → ³¹P + p
           → ³¹S + n
```

### 3.4 硅燃烧与核统计平衡

**光解离-重组平衡**：
```
γ + ²⁸Si ⇌ ⁴He + ²⁴Mg
```

形成铁峰元素（V, Cr, Mn, Fe, Co, Ni）：
- 这些核素有最高的结合能 per nucleon
- 能量产生停止
- 恒星核心坍缩

## 4. 超新星核合成

### 4.1 核坍缩超新星

**爆发机制**：
1. 铁核坍缩 → 中子星
2. 反弹激波
3. 中微子加热
4. 物质抛出

**r-过程场所**：
- 传统观点：超新星中微子球层
- 新观点：中子星并合（主要场所）

### 4.2 r-过程（快中子捕获）

**条件**：
- 高中子密度：n_n > 10²⁰ cm⁻³
- 高温：T ~ 10⁹ K
- 快速捕获：τ_n < τ_β

**反应链**：
```
(A,Z) + n → (A+1,Z)
(A+1,Z) + n → (A+2,Z) 或 → (A+1,Z+1) + e⁻ + ν̄_e
```

**路径**：
沿中子滴线进行，直到β衰变率超过中子捕获率。

**r-过程峰**：
- 第一峰：A ~ 80（Se, Br, Kr）
- 第二峰：A ~ 130（Te, I, Xe）
- 第三峰：A ~ 195（Pt, Au, Os）

对应幻数：N = 50, 82, 126

### 4.3 s-过程（慢中子捕获）

**场所**：渐近巨星分支（AGB）星

**条件**：
- 低中子密度：n_n ~ 10⁸ cm⁻³
- τ_n >> τ_β

**中子源**：
```
¹³C(α,n)¹⁶O
²²Ne(α,n)²⁵Mg
```

**s-过程路径**：
沿稳定谷进行，每次β衰变后捕获中子。

**s-过程丰度**：
平滑分布，无尖锐峰。

## 5. 宇宙射线散裂与轻元素

### 5.1 Li-Be-B的起源

**问题**：BBN产生少量Li，但观测到的Li-Be-B更多。

**机制**：
宇宙射线（C, N, O, Fe）与星际介质（H, He）碰撞：
```
¹²C + p → ¹¹B + 2p
¹²C + p → ¹⁰B + p + n
¹²C + p → ⁹Be + ³He
```

### 5.2 计算模型

**传播方程**：
```
dN_LiBeB/dt = Σ_{CNO} ∫ σ_{spall}(E) φ(E) dE - Λ_{destr}N_LiBeB
```

**结果**：
成功解释Li-Be-B的观测丰度。

## 6. 元素丰度的宇宙学演化

### 6.1 金属丰度随时间增加

**定义**：
```
[Fe/H] = log(N_Fe/N_H) - log(N_Fe/N_H)_⊙
```

**观测趋势**：
- 早期星系：[Fe/H] < -3
- 太阳年龄：[Fe/H] = 0
- 现在：[Fe/H] ~ +0.1

### 6.2 化学演化模型

**简单封闭盒模型**：
```
dZ/dt = y(1-Z)
```

其中：
- Z：金属丰度
- y：产额（每代恒星产生的金属）

**解**：
```
Z(t) = 1 - exp(-yt)
```

### 6.3 银河系化学演化

**观测约束**：
- 恒星金属丰度分布
- 气体金属丰度梯度
- 年龄-金属丰度关系

**模型参数**：
- 恒星形成历史
- 初始质量函数
- 核合成产额
- 气体流入/流出

## 7. 同位素丰度与核过程指纹

### 7.1 同位素比作为探针

**示例**：
- ¹²C/¹³C：区分s-过程和CNO循环
- ²⁰⁶Pb/²⁰⁷Pb：测定恒星年龄
- ²⁶Mg/²⁴Mg：检测灭绝的²⁶Al

### 7.2 前太阳系颗粒

**来源**：
- 超新星
- AGB星
- 新星

**信息**：
- 异常同位素组成
- 原始太阳星云的混合
- 恒星核合成的细节

### 7.3 短寿命放射性核素

**灭绝核素证据**：
- ²⁶Al（τ_{1/2} = 0.7 Myr）
- ⁶⁰Fe（τ_{1/2} = 2.6 Myr）
- ¹⁰⁷Pd（τ_{1/2} = 6.5 Myr）

**意义**：
太阳系形成前附近发生过超新星或AGB星。

## 8. 元素起源的未解问题

### 8.1 轻元素问题

**锂缺失**：
BBN预言 vs 观测恒星值差异3倍

**可能的解决**：
- 早期恒星锂消耗
- BBN期间的非标准物理
- 观测选择效应

### 8.2 重元素问题

**r-过程场所**：
- 超新星？中子星并合？
- GW170817确认了中子星并合产生r-过程元素
- 但可能不是唯一场所

**p-过程（富质子同位素）**：
- 超新星爆炸？
- 热核爆炸？
- 机制不完全清楚

### 8.3 宇宙学问题

**第一代恒星（星族III）**：
- 零金属丰度
- 质量极大（~100 M_⊙）
- 从未被直接观测到

**暗物质与元素合成**：
- 原初黑洞？
- 暗物质衰变？
- 可能影响早期核合成
