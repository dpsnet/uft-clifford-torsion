# 数学推导笔记：暗物质候选者——量子化遗迹

## 1. 引言：黑洞信息悖论与量子遗迹

### 1.1 问题背景

标准黑洞热力学预言黑洞通过霍金辐射完全蒸发，但这就产生了**信息悖论**：
- 纯态坍缩为黑洞
- 黑洞蒸发为热辐射（混合态）
- 幺正性被破坏

**本理论的解决方案**：
- 黑洞内部没有奇点，而是扭转饱和的分形核心
- 信息编码在扭转量子态中
- 黑洞蒸发停止于普朗克质量尺度，留下**量子遗迹**

### 1.2 量子遗迹假说

**核心观点**：
黑洞蒸发不完全——当黑洞质量降至普朗克质量量级时，扭转势垒阻止进一步蒸发，形成稳定的量子遗迹。

**物理动机**：
1. 信息守恒：遗迹携带所有原始信息
2. 幺正性保持：整体演化保持量子相干性
3. 暗物质候选：遗迹有恰当的质量和丰度

## 2. 遗迹形成机制

### 2.1 扭转饱和机制

**定理**：当黑洞质量降至临界值 M_crit，扭转场饱和阻止霍金辐射。

**推导**：

黑洞内部扭转强度与质量的关系：
```
τ(M) = τ_max (1 - M/M_Planck)^{-α}
```

其中：
- τ_max ~ 1（最大扭转强度）
- α ~ 1（模型参数）
- M_Planck = √(ℏc/G) ~ 10⁻⁸ kg

当 M → M_Planck，τ → ∞，几何效应主导。

**蒸发停止条件**：

霍金温度：
```
T_H = ℏc³/(8πGMk_B)
```

当 T_H ~ E_Planck = M_Planck c²，蒸发停止。

这发生在：
```
M_crit = M_Planck / √(8π) ~ 2 × 10⁻⁹ kg
```

### 2.2 遗迹的质量分布

**质量函数**：

遗迹质量分布取决于原初黑洞形成机制：

**原初黑洞形成**：
早期宇宙高扭转区域坍缩形成原初黑洞：
```
M_PBH = γ (c_s²/G) t
```

其中 γ ~ 0.2，c_s 是声速，t 是形成时间。

**遗迹质量谱**：
```
dn/dM_remnant ∝ (M_PBH/M_crit)^{-2.5} for M_PBH > M_crit
```

对于蒸发到当前的遗迹：
```
dn/dM ~ M^{-1/2} for M ~ M_crit
```

**数值范围**：
- 遗迹质量：M_remnant ~ 10⁻⁹ - 10⁻⁶ kg
- 对应能量：E ~ 10⁹ - 10¹² GeV
- 粒子数密度：n ~ 10⁹ m⁻³（如果占暗物质100%）

### 2.3 遗迹的内部结构

**分形核心**：

遗迹内部是**扭转饱和的分形结构**：
- 谱维度 D_s ≈ 2
-  Hausdorff维度 D_H ≈ 2.5
- 信息存储在分形表面的量子态中

**信息容量**：

Bekenstein-Hawking熵的修正：
```
S_remnant = k_B (A/4L_P²) + S_{torsion}
```

其中扭转贡献：
```
S_{torsion} ~ k_B τ² (A/L_P²)^{D_s/4}
```

对于 D_s = 2，S_{torsion} ~ k_B τ² √(A/L_P²)

总信息容量足以编码原初黑洞的所有信息。

## 3. 遗迹作为暗物质候选者

### 3.1 观测约束

**暗物质的基本性质**：
- 不发光（无电磁相互作用）
- 有引力效应
- 非相对论性（冷暗物质）
- 稳定（寿命 > 宇宙年龄）

**遗迹满足所有条件**：

| 性质 | 遗迹行为 | 是否满足 |
|-----|---------|---------|
| 无电磁作用 | 仅有引力相互作用 | ✓ |
| 引力效应 | 有质量 | ✓ |
| 冷暗物质 | M ~ 10⁻⁹ kg，v ~ 10⁻³ c | ✓ |
| 稳定性 | 扭转势垒保护 | ✓ |

### 3.2 丰度计算

**遗迹数密度**：

如果遗迹构成全部暗物质：
```
Ω_remnant = Ω_DM ≈ 0.26
```

数密度：
```
n_remnant = ρ_DM / M_remnant
          ≈ (0.26 × 10⁻²⁶ kg/m³) / (10⁻⁹ kg)
          ≈ 10¹⁷ m⁻³
```

**原初黑洞形成率**：

为了产生足够遗迹，原初黑洞必须足够丰富：
```
β = ρ_PBH / ρ_total at formation
```

需要 β ~ 10⁻⁸ - 10⁻⁵（取决于质量谱）。

### 3.3 质量窗口

**约束条件**：

1. **微引力透镜**（EROS、OGLE、Subaru）：
   - 排除 10²⁴ kg < M < 10³⁰ kg
   - 遗迹质量 ~10⁻⁹ kg 不在此范围

2. **宇宙学约束**：
   - 遗迹质量太小不显著影响结构形成
   - 符合冷暗物质模型

3. **稳定性约束**：
   - M > M_Planck 保证遗迹稳定

**允许的质量窗口**：
```
10⁻⁹ kg < M_remnant < 10⁻⁶ kg
```

对应能量：

## 4. 遗迹的探测方案

### 4.1 引力效应

**遗迹的引力透镜**：

虽然单个遗迹质量小，但大量遗迹产生集体引力效应。

透镜效应：
```
θ_E = √(4GM_remnant D_LS / c² D_L D_S)
```

对于 M ~ 10⁻⁹ kg，θ_E ~ 10⁻²⁰ arcsec（不可观测）。

**集体效应**：
遗迹云产生的密度扰动影响：
- 星系旋转曲线
- 引力透镜统计
- 结构形成

与标准冷暗物质不可区分。

### 4.2 碰撞效应

**遗迹-核子碰撞**：

虽然遗迹仅通过引力相互作用，但在极高能下可能有微弱耦合。

弹性散射截面：
```
σ ~ G² M² / v⁴ ~ 10⁻⁶⁰ m²
```

远小于当前直接探测实验灵敏度（σ > 10⁻⁴⁸ m²）。

### 4.3 信号特征

**原初黑洞并合率**：

如果遗迹来自原初黑洞，原初黑洞并合产生引力波背景：
```
Ω_GW(f) ~ 10⁻⁹ (f/1 Hz)^{2/3}
```

NANOGrav观测到的背景可能与原初黑洞并合有关。

**遗迹分布特征**：

遗迹形成时的不均匀分布可能在今天产生特征：
- 小尺度功率谱增强
- 原初非高斯性

### 6.1 遗迹稳定性证明

**定理**：扭转势垒保证遗迹稳定。

**证明概要**：

1. 扭转势能在 τ → ∞ 时有最小值
2. 霍金辐射需要 τ 有限
3. 当 τ → ∞，辐射率 → 0
4. 遗迹是基态，无法进一步衰变

### 6.2 信息守恒

**定理**：遗迹编码原初黑洞的全部信息。

```

```

对于原初黑洞 M ~ 10⁻⁶ kg：
```
N_bits ~ 10⁴⁰
```

足以编码所有落入黑洞的信息。

### 6.3 幺正性保持

**定理**：黑洞蒸发+遗迹形成过程保持幺正性。

**论证**：
1. 初始纯态 |ψ⟩

   - 霍金辐射（混合态）
