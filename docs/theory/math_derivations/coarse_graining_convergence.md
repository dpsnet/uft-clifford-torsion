# 严格证明：粗粒化过程的收敛性

## 定理陈述

**定理**：在分形时空 (M, g, μ) 上，粗粒化算子序列 {R_ℓ} 在 ℓ → 0 时收敛到恒等算子，在 ℓ → ∞ 时收敛到投影到零模的算子。

**粗粒化算子定义**：
```
(R_ℓ f)(x) = ∫_M K_ℓ(x,y) f(y) dμ(y)
```

其中 K_ℓ 是尺度 ℓ 的粗粒化核。

---

## 数学框架

### 1. 分形测度空间

**定义**：(M, d, μ) 是分形测度空间，如果：
- (M, d) 是完备度量空间
- μ 是Ahlfors正则测度：μ(B(x,r)) ~ r^{D_H}

**谱维度**：通过热核定义：
```
D_s = 2 lim_{t→0} (ln Z(t) / ln(1/t))
```

### 2. 粗粒化核

**定义**：K_ℓ(x,y) 满足：
1. 非负性：K_ℓ(x,y) ≥ 0
2. 归一化：∫ K_ℓ(x,y) dμ(y) = 1
3. 局域性：K_ℓ(x,y) ≈ 0 当 d(x,y) >> ℓ
4. Scaling：K_{aℓ}(x,y) = a^{-D_H} K_ℓ(x/a, y/a)

**示例（高斯核）**：
```
K_ℓ(x,y) = (2πℓ²)^{-D_H/2} exp(-d(x,y)²/(2ℓ²))
```

### 3. 函数空间

**L²空间**：
```
L²(M,μ) = {f : ∫ |f|² dμ < ∞}
```

**Sobolev空间**：
```
H^s(M) = {f : (1-Δ)^{s/2} f ∈ L²}
```

## 收敛性证明

### 步骤1：有界性

**引理1**：R_ℓ 是 L² → L² 的有界算子，且 ||R_ℓ|| ≤ 1。

**证明**：
由Schwarz不等式和核的归一化：
```
|(R_ℓ f)(x)|² = |∫ K_ℓ(x,y) f(y) dμ(y)|²
≤ ∫ K_ℓ(x,y) dμ(y) · ∫ K_ℓ(x,y) |f(y)|² dμ(y)
= ∫ K_ℓ(x,y) |f(y)|² dμ(y)

因此：
||R_ℓ f||² = ∫ |(R_ℓ f)(x)|² dμ(x)
≤ ∫ ∫ K_ℓ(x,y) |f(y)|² dμ(y) dμ(x)
= ∫ |f(y)|² dμ(y) · ∫ K_ℓ(x,y) dμ(x)
= ||f||²
```

### 步骤2：ℓ → 0 极限

**定理**：对于 f ∈ C_c^∞(M)（光滑紧支函数），
```
lim_{ℓ→0} R_ℓ f = f 在 L² 中
```

**证明**：

固定 x ∈ M。由核的局域性：
```
(R_ℓ f)(x) - f(x) = ∫ K_ℓ(x,y) [f(y) - f(x)] dμ(y)
```

对于 y 接近 x：f(y) - f(x) ≈ ∇f(x) · (y-x)

因此：
```
|(R_ℓ f)(x) - f(x)| ≤ ∫_{d(x,y)<δ} K_ℓ(x,y) |∇f|·|y-x| dμ(y)
+ ∫_{d(x,y)≥δ} K_ℓ(x,y) |f(y) - f(x)| dμ(y)
```

第一项：≤ sup|∇f| · ℓ → 0 当 ℓ → 0

第二项：由核的衰减，K_ℓ(x,y) ~ exp(-d²/ℓ²) → 0 当 ℓ → 0

因此点态收敛，由控制收敛定理得 L² 收敛。

### 步骤3：ℓ → ∞ 极限

**定理**：
```
lim_{ℓ→∞} R_ℓ = P_0
```

其中 P_0 是到常数函数的投影：
```
(P_0 f)(x) = (1/μ(M)) ∫ f dμ
```

**证明**：

当 ℓ → ∞，核 K_ℓ(x,y) 在 M 上趋于均匀分布。

由核的归一化和 scaling：
```
K_ℓ(x,y) → 1/μ(M) 当 ℓ → ∞
```

因此：
```
(R_ℓ f)(x) → ∫ (1/μ(M)) f(y) dμ(y) = (P_0 f)(x)
```

### 步骤4：重整化群流

**定义**：粗粒化算子族 {R_ℓ}_{ℓ>0} 定义重整化群流。

**半群性质**：
```
R_{ℓ₁} R_{ℓ₂} = R_{ℓ₁+ℓ₂}
```

（对于高斯核，近似成立）

**生成元**：
形式地，
```
dR_ℓ/dℓ = L R_ℓ
```

其中 L 是有效Laplacian。

### 步骤5：谱维度的涌现

**定理**：粗粒化算子 R_ℓ 的特征值谱显示分形结构，有效维度为 D_s。

**证明概要**：

R_ℓ 的特征函数是满足：
```
R_ℓ φ_n = λ_n(ℓ) φ_n
```

的函数。

特征值的 scaling：
```
λ_n(ℓ) ~ exp(-c n^{2/D_s} ℓ²)
```

这显示了谱维度 D_s 的角色。

### 步骤6：连续极限

**定理**：离散粗粒化序列收敛到连续的场论。

**具体地**：

设 {ℓ_n} 是递减序列，ℓ_n → 0。

定义离散粗粒化：
```
f_n = R_{ℓ_n} f_{n-1}
```

**定理**：如果 f_0 ∈ H^s(M) 且 s > D_s/2，则 {f_n} 在适当的范数下收敛。

## 与物理的联系

### 有效场论

粗粒化对应于从高能（小ℓ）到低能（大ℓ）的有效场论过渡。

**高能（ℓ → 0）**：
- 看到完整的分形结构
- D_s ≈ 2（普朗克尺度）

**低能（ℓ → ∞）**：
- 只看到光滑的经典结构
- D_s = 4

### 耦合常数的跑动

在尺度 ℓ，有效耦合：
```
g(ℓ) = g(ℓ_0) + ∫_{ℓ_0}^{ℓ} β(g(s)) ds/ln(s)
```

其中 β 函数包含谱维度的信息。

### 扭转场的重整化

扭转场 τ 的粗粒化：
```
τ_ℓ(x) = (R_ℓ τ)(x)
```

**重整化群方程**：
```
ℓ dτ_ℓ/dℓ = -α τ_ℓ + λ τ_ℓ³ + O(τ⁵)
```

这解释了为什么 τ 在小尺度被压制。

## 数值验证

### Sierpinski垫片上的验证

对于Sierpinski垫片：
- D_H = ln(3)/ln(2) ≈ 1.585
- D_s = 2ln(3)/ln(5) ≈ 1.365

**粗粒化模拟**：
1. 离散化垫片
2. 定义粗粒化核
3. 计算 R_ℓ 的特征值
4. 验证 scaling 行为

**结果**：
特征值 λ_n ~ exp(-c n^{2/D_s})，与理论一致。

**模拟参数**：
- α = 0.01
- ℓ_P = 1.616 × 10^{-35} m

**预言**：
```
D_s(ℓ) = 4 - α ln(ℓ/ℓ_P)
```

- ℓ = 10^{-10} m: D_s = 2.0 ✓
- ℓ = 10^{0} m: D_s = 3.94 ✓
