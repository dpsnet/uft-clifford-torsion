## 4. 证明策略

### 4.1 第一步：世界面 ↔ 内部空间

**目标**: 证明弦世界面 = 内部空间的2维子流形

**方法**:
- 弦世界面: $(\tau, \sigma)$ 参数化
- 内部空间: 谱维 $d_s$ 随能量变化
- 在特定能量下，内部空间"展开"出弦世界面

### 4.2 第二步：弦振动 ↔ 扭转激发

**目标**: 弦振动模式 = 扭转场本征模式

**方法**:
- 弦: $X^\mu = x^\mu + \sum_n \alpha_n^\mu e^{-in\sigma}$
- 扭转: $\tau_{\mu\nu\rho} = \sum_k \tau_k \phi_k(x)$
- 识别: $\alpha_n^\mu \leftrightarrow \tau_k$

### 4.3 第三步：作用量等价

**目标**: 弦作用量 = 扭转作用量（低能有效理论）

**方法**:
- 弦低能有效作用: 超引力
- 扭转有效作用: Einstein-Cartan + 物质场
- 在 $d_s \to 10$ 极限下证明等价
