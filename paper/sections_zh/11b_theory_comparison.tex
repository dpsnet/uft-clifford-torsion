\section{理论比较}
\label{sec:theory_comparison}

CTUFT与其他量子引力理论的比较。

\subsection{弦理论}

\begin{itemize}
    \item 额外维度：6维紧致化（弦理论）vs. 6维谱维度（CTUFT）
    \item 自由参数：多个（弦理论）vs. 零个（CTUFT）
    \item 可检验性：困难（弦理论）vs. 可证伪（CTUFT）
\end{itemize}

\subsection{圈量子引力}

\begin{itemize}
    \item 分立性：基本的（LQG）vs. 涌现的（CTUFT）
    \item 黑洞熵：相似的对数修正
\end{itemize}
