\section{实验验证与可证伪预测}
\label{sec:experimental}

\begin{abstract}
本章介绍克利福德-扭转统一场论（CTUFT）的具体、可实验检验的预言。我们聚焦于：
(1) 引力波探测：六种偏振模式、修正的传播速度；
(2) 原初黑洞（PBH）霍金辐射：作为检验谱维流动 $d_s \to 10$ 的探针；
(3) 五项具体的可证伪预测，涵盖从CTA可探测的伽马射线谱修正（2025+）到LISA可观测的引力波信号（2030s）。
这些预测将CTUFT从一个纯理论框架转变为具有实验可检验性的物理理论。
\end{abstract}

\subsection{可检验预言概述}

我们的克利福德-扭转统一场论（CTUFT）做出了几个具体的、可证伪的预言，这些预言将其与标准物理区分开来。这些预言分为四类：

\begin{enumerate}
    \item \textbf{引力波检验}：六种偏振模式、修正的传播速度和特征波形
    \item \textbf{原初黑洞检验}：霍金辐射谱修正、质量谱特征峰
    \item \textbf{粒子物理检验}：修正的原子光谱、耦合常数关系和质量公式
    \item \textbf{宇宙学检验}：原初引力波谱、暗物质丰度和大尺度结构
\end{enumerate}

单一参数 $\tau_0 \approx 0.005$ 使用信息论从第一性原理导出，代表10维信息投影到4维的保真度。

%==============================================================================
\subsection{引力波探测}
%==============================================================================

\subsubsection{LISA：主要探测通道}

激光干涉空间天线（LISA），计划于2034年发射，是我们的主要探测通道。LISA对中等质量黑洞并合（$10^4$--$10^7 M_\odot$）的灵敏度为检验我们的预言提供了理想频率范围。

\textbf{LISA的关键预言}：

\begin{itemize}
    \item \textbf{六种偏振模式}：标准广义相对论预言两种（$h_+$、$h_\times$）；我们预言六种，包括矢量（$h_x$、$h_y$）和标量（$h_b$、$h_l$）模式
    \item \textbf{相对振幅}：
    \begin{align}
        \frac{h_x}{h_+} \sim \frac{h_y}{h_+} \sim \frac{\tau_0}{2} \approx 2.5 \times 10^{-3} \\
        \frac{h_b}{h_+} \sim \frac{\tau_0^2}{3} \approx 8 \times 10^{-6}
    \end{align}
    
    \item \textbf{色散传播}：不同偏振以略微不同速度传播，在宇宙距离上导致到达时间延迟
\end{itemize}

\textbf{探测策略}：

LISA的三角构型，三个航天器相隔250万公里，提供探测额外偏振所需的灵敏度。距离1 Gpc的 $10^5 M_\odot$ 双星的信噪比为：

\begin{equation}
    \text{SNR} \approx 1000 \quad \text{（对于 } \tau_0 = 0.005\text{）}
\end{equation}

这显著高于标准偏振的信噪比，使探测极有可能成功。

\subsubsection{脉冲星计时阵列}

虽然脉冲星计时阵列（NANOGrav、EPTA、PPTA）对nHz--$\mu$Hz范围的引力波敏感，但来自超大质量黑洞双星的随机背景对我们的预言效应占主导（$\tau_0 = 0.005$）。

\begin{equation}
    \frac{h_{CTUFT}}{h_{astrophysical}} \sim 10^{-4} \quad \text{（未来PTA可探测）}
\end{equation}

具有改进计时精度的IPTA（国际脉冲星计时阵列）将在2030年代达到检验 $\tau_0 = 0.005$ 的灵敏度。

\subsection{原初黑洞：检验谱维流动}
%==============================================================================

\subsubsection{作为探针的原初黑洞}

早期宇宙中形成的原初黑洞（PBHs）为检验CTUFT提供了理想实验室：

\begin{itemize}
    \item \textbf{蒸发时标}：质量 $M \sim 10^{15}$ g的PBH正在今天蒸发，发出可观测辐射
    \item \textbf{霍金温度}：$M \sim 10^{15}$ g时 $T_H \sim 10$ MeV，产生伽马射线和正电子
    \item \textbf{与$\tau_0$的关联}：蒸发PBH的特征质量尺度与基本参数$\tau_0$相联系
\end{itemize}

\subsubsection{CTUFT修正的霍金辐射}

\textbf{标准霍金辐射}：
\begin{equation}
n_{std}(E) = \frac{1}{e^{E/T_H} - 1}
\end{equation}

\textbf{CTUFT修正谱}（包含谱维流动效应）：
\begin{equation}
n_{CTUFT}(E) = n_{std}(E) \times \mathcal{F}(E)
\end{equation}

其中\textbf{CTUFT修正因子}为：
\begin{equation}
\mathcal{F}(E) = \left(\frac{E}{E_c}\right)^{\alpha(E)}, \quad \alpha(E) = \frac{d_s(E)}{2} - 2
\end{equation}

谱维流动公式：
\begin{equation}
d_s(E) = 4 + \frac{6}{1 + (E/E_c)^2} \rightarrow 10 \text{（高能极限）}
\end{equation}

\textbf{关键差异}：CTUFT预测在 $E \gtrsim 5 T_H$ 处有显著增强的高能尾。

\subsection{五项可证伪预测}
%==============================================================================

\subsubsection{预测1：伽马射线谱增强}

\textbf{观测}：切伦科夫望远镜阵列（CTA，2025+年运行）

\textbf{预测}：
\begin{itemize}
    \item 能量范围：$E \gtrsim 5 \times T_H \sim 50$ MeV（对于 $M \sim 10^{15}$ g PBH）
    \item 信号：与标准霍金谱相比增强的伽马射线尾
    \item 幅度：$E \sim 10 \times T_H$ 处增强2--5倍
\end{itemize}

\textbf{当前约束}：Fermi-LAT CGB限制 $f_{PBH} < 10^{-8}$

\textbf{CTA改进}：对 $f_{PBH} \sim 10^{-10}$ 的灵敏度（100倍提升）

\textbf{证伪判据}：如果CTA未观测到超出或谱形与CTUFT预测不符

\subsubsection{预测2：PBH质量谱峰}

\textbf{观测}：微引力透镜巡天（Subaru HSC、OGLE、下一代）

\textbf{预测}：
\begin{itemize}
    \item 特征质量：$M_{peak} \sim 10^{15}$ g
    \item 起源：通过蒸发时标与 $\tau_0 = 0.005$ 相联系
    \item 谱形：窄峰而非宽分布
\end{itemize}

\textbf{物理关联}：
\begin{equation}
t_{evap} = \frac{5120\pi G^2 M^3}{\hbar c^4} \approx 13.8 \text{ Gyr（对于 } M \sim 10^{15} \text{ g）}
\end{equation}

\textbf{证伪判据}：如果微透镜巡天在 $M \sim 10^{15}$ g处未发现峰

\subsubsection{预测3：正电子谱修正}

\textbf{观测}：AMS-02、DAMPE（空间宇宙线探测器）

\textbf{预测}：
\begin{itemize}
    \item 能量范围：$E \gtrsim 10$ GeV
    \item 信号：来自PBH蒸发的过量正电子
    \item 谱形：被 $d_s \to 10$ 增强所修正
\end{itemize}

\textbf{当前状态}：AMS-02在 $E \gtrsim 10$ GeV观测到正电子超出

\textbf{证伪判据}：如果高分辨率数据排除CTUFT谱形

\subsubsection{预测4：PBH并合率（LISA）}

\textbf{观测}：LISA空间引力波探测器（2030s年代）

\textbf{预测}：
\begin{itemize}
    \item 质量范围：$10^{-12} - 10^{-7}$ M$_\odot$（LISA敏感）
    \item 并合率：依赖于PBH丰度 $f_{PBH}$
    \item 可探测性：暗物质的 $f_{PBH} \gtrsim 10^{-3}$ 部分
\end{itemize}

\textbf{证伪判据}：如果PBH构成暗物质的 $f \gtrsim 10^{-3}$ 但未探测到并合信号

\subsubsection{预测5：多信使一致性}

\textbf{观测}：伽马射线、中微子、引力波数据的联合分析

\textbf{预测}：
\begin{itemize}
    \item 相同PBH群体在所有波段产生信号
    \item 独立测量的PBH丰度 $f_{PBH}$ 一致
    \item 所有通道都有CTUFT修正谱
\end{itemize}

\textbf{关键检验}：CTA伽马射线、IceCube-Gen2中微子、LISA引力波的交叉关联

\textbf{证伪判据}：如果不同信使给出不一致的PBH性质

    \item \textbf{Subaru HSC}：$f_{PBH} < 0.01$ 对于 $10^{11} - 10^{17}$ g
    \item \textbf{EROS}：$f_{PBH} < 0.1$ 对于 $2\times10^{-7} - 10^{-4}$ M$_\odot$
    \item \textbf{OGLE}：$f_{PBH} < 0.1$ 对于 $10^{-6} - 10^{-3}$ M$_\odot$
\end{itemize}

\subsubsection{原子钟约束}

原子钟为 $\tau_0$ 提供了最严格的约束。挠率场诱导频率偏移：

\begin{equation}
    \frac{\Delta \nu}{\nu} \sim \tau_0 \cdot g_\tau \cdot 10^{11}
\end{equation}

当前光晶格钟达到 $\Delta\nu/\nu \sim 10^{-18}$ 的稳定性，约束：

\begin{equation}
    \tau_0 \lesssim 0.01
\end{equation}

我们导出的值 $\tau_0 = 0.005$ 在当前约束范围内。

\subsection{结论}
%==============================================================================

CTUFT做出了五项具体的、实验可检验的预言：

\begin{enumerate}
    \item \textbf{引力波偏振}：6种模式，LISA可探测
    \item \textbf{伽马射线增强}：CTA可探测（2025+）
    \item \textbf{质量谱峰}：微引力透镜可检验
    \item \textbf{正电子/中微子}：多信使一致性
    \item \textbf{PBH并合}：LISA 2030s探测
\end{enumerate}

\textbf{关键信息}：CTUFT不仅仅是一个理论框架，而是一个将在未来10--15年内被观测验证或证伪的\textbf{物理可检验理论}。
