\section{引言}
\label{sec:introduction}

\subsection{寻求统一}

基本力的统一自20世纪初以来一直是理论物理学的圣杯。爱因斯坦的广义相对论提供了引力的几何描述，而量子场论成功地描述了电磁力、弱力和强力。然而，现代物理学的这两根支柱在量子引力尺度上仍然根本上不相容。

粒子物理学的标准模型虽然在描述粒子及其相互作用方面取得了显著成功，但留下了几个深刻的问题未解答：
\begin{itemize}
    \item 为什么存在三代费米子，其质量和混合角看似任意？
    \item 规范对称性 $U(1) \times SU(2) \times SU(3)$ 的起源是什么？
    \item 量子力学和广义相对论如何协调？
    \item 黑洞和大爆炸中的奇点如何解决？
    \item 暗物质和暗能量的本质是什么？
\end{itemize}

\subsection{现有方法及其局限性}

已提出多种方法来解决这些问题：

\textbf{弦理论}提出基本实体是在10或11维中振动的一维弦。虽然在数学上优雅且具有潜在的统一性，但弦理论面临诸多挑战，包括景观问题、缺乏实验预言，以及在可及能量处缺乏超对称性。

\textbf{圈量子引力}试图将时空本身量子化，产生面积和体积算符的离散谱。然而，它难以纳入标准模型规范力，并面临关于洛伦兹不变性的问题。

\textbf{非交换几何}提供了一种统一的代数方法，从谱三元组导出标准模型。虽然在再现粒子内容方面取得成功，但它缺乏引力和量子力学的清晰几何解释。

\textbf{因果集理论}和其他离散方法假设时空的基本离散性，但在适当极限下难以恢复连续的广义相对论。

\subsection{本方法}

本工作提出了一种基于几个关键创新的统一场论：

\begin{enumerate}
    \item \textbf{固定4D拓扑与动态谱维数}：我们将拓扑维数保持在 $d_t = 4$，同时允许谱维数 $\ds$ 从高能时的10动态变化到低能时的4。这解决了景观问题，同时保留了高维理论的优势。
    
    \item \textbf{分形克利福德代数}：代数结构自然地纳入分形测度，为描述时空的连续和离散方面提供了框架。
    
    \item \textbf{多重扭曲机制}：规范对称性的几何起源来自覆盖映射中的独立 $Z_2$ 因子，对应于时空纤维的不同扭曲自由度。
    
    \item \textbf{倒易-内部空间对偶性}：可观测时空（倒易空间）与内部自由度之间的基本对偶性为理解量子纠缠和信息流提供了自然框架。
\end{enumerate}

\subsection{关键结果与预言}

我们的理论产生了几个具体的、可证伪的预言：

\begin{itemize}
    \item \textbf{六种引力波偏振模式}：与广义相对论的两种偏振模式（$+$ 和 $\times$）不同，我们的理论预言了六种模式，包括矢量偏振和标量偏振。LISA和下一代地面探测器可以搜索这些模式。
    
    \item \textbf{几何量子力学}：量子力学自然地从时空拓扑中产生，消除了测量问题，并为波函数坍缩提供了几何解释。
    
    \item \textbf{黑洞奇点的解决}：黑洞内部包含扭曲饱和的分形核心而非奇点，通过量子化的扭曲模式解决信息悖论。
    
    \item \textbf{暗物质候选者}：该理论预言的量子黑洞残余自然地提供了具有正确丰度的暗物质候选者。
\end{itemize}

单一参数 $\tau_0 \approx 0.005$ 是使用信息论从第一性原理导出的（参见 \cref{app:tau_zero_derivation}），代表10维信息投影到4维的保真度。该参数控制扭曲效应的强度，并为所有实验检验提供定量预言。

\subsection{符号和约定}

在本文中，我们使用以下约定：
\begin{itemize}
    \item 度规符号：$(-,+,+,+)$
    \item 自然单位：$\hbar = c = G = 1$（在适当处）
    \item 克利福德代数生成元：$\{\gamma^\mu, \gamma^\nu\} = 2g^{\mu\nu}$
    \item 挠率张量：$T^\lambda{}_{\mu\nu} = \Gamma^\lambda{}_{\mu\nu} - \Gamma^\lambda{}_{\nu\mu}$
    \item 谱维数：$\ds = -2 \frac{d \ln Z(t)}{d \ln t}$，其中 $Z(t) = \Tr(e^{t\Delta})$
\end{itemize}

希腊指标 $\mu, \nu, \ldots$ 遍历时空维数 $0,1,2,3$。拉丁指标 $a,b,\ldots$ 表示内部框架指标。大写字母 $I,J,\ldots$ 在需要时标记内部空间维数。
