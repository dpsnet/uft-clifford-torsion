\section{实验验证与可证伪预测}
\label{sec:experimental}

\begin{insight}[本章概要]
本章介绍克利福德-扭转统一场论（CTUFT）的具体、可实验检验的预言。我们聚焦于原初黑洞（PBH）霍金辐射作为检验理论谱维流动 $d_s \to 10$ 的探针。五项具体的可证伪预测被导出，涵盖从CTA可探测的伽马射线谱修正（2025+）到LISA可观测的引力波信号（2030s）。这些预测将CTUFT从一个纯理论框架转变为具有实验可检验性的物理理论。
\end{insight}

\subsection{引言：从理论到实验}
\label{subsec:exp-intro}

任何物理理论的最终检验在于其实验预言。尽管CTUFT在数学上相当复杂，但它对可观测现象做出了具体、可证伪的陈述。本节将第\ref{ch:mathematical}--\ref{ch:black-holes}章发展的理论框架与实验物理联系起来。

\subsubsection{作为可观测量的谱维}

CTUFT的核心新特征——谱维流动 $d_s = 4 \to 10$——不仅仅是数学构造，而是具有可观测后果的物理现象：

\begin{equation}
d_s(E) = 4 + \frac{6}{1 + (E/E_c)^2}
\label{eq:spectral-flow-exp-zh}
\end{equation}

其中 $E_c \sim \tau_0 \times m_P c^2$ 是由单一自由参数 $\tau_0 = 0.005$ 设定的特征能量尺度。

\textbf{关键洞见}：高能极限 $d_s \to 10$ 修正了量子场论中的态密度，导致黑洞辐射谱的高能尾增强。

\subsubsection{作为探针的原初黑洞}

早期宇宙中形成的原初黑洞（PBHs）为检验CTUFT提供了理想实验室：

\begin{itemize}
    \item \textbf{蒸发时标}：质量 $M \sim 10^{15}$ g的PBH正在今天蒸发，发出可观测辐射
    \item \textbf{霍金温度}：$M \sim 10^{15}$ g时 $T_H \sim 10$ MeV，产生伽马射线和正电子
    \item \textbf{与$\tau_0$的关联}：蒸发PBH的特征质量尺度与基本参数$\tau_0$相联系
\end{itemize}

\subsection{CTUFT修正的霍金辐射}
\label{sec:modified-hawking-zh}

\subsubsection{标准与CTUFT辐射谱对比}

\textbf{标准霍金辐射}：
\begin{equation}
n_{std}(E) = \frac{1}{e^{E/T_H} - 1}
\label{eq:standard-hawking-zh}
\end{equation}

\textbf{CTUFT修正谱}（包含谱维流动效应）：
\begin{equation}
n_{CTUFT}(E) = n_{std}(E) \times \mathcal{F}(E)
\label{eq:ctuft-hawking-zh}
\end{equation}

其中\textbf{CTUFT修正因子}为：
\begin{equation}
\mathcal{F}(E) = \left(\frac{E}{E_c}\right)^{\alpha(E)}, \quad \alpha(E) = \frac{d_s(E)}{2} - 2
\label{eq:correction-factor-zh}
\end{equation}

\subsubsection{物理解释}

修正因子 $\mathcal{F}(E)$ 来源于：
\begin{enumerate}
    \item \textbf{态密度}：在 $d_s$ 维中，$\rho(E) \sim E^{d_s/2 - 1}$
    \item \textbf{相空间}：更高的 $d_s$ 增加了高能处的可用态
    \item \textbf{有效温度}：谱流动有效地修正了辐射温度
\end{enumerate}

\subsection{五项可证伪预测}
\label{sec:falsifiable-predictions-zh}

\subsubsection{预测1：伽马射线谱增强}
\label{subsec:prediction-gamma-zh}

\textbf{观测}：切伦科夫望远镜阵列（CTA，2025+年运行）

\textbf{预测}：
\begin{itemize}
    \item 能量范围：$E \gtrsim 5 \times T_H \sim 50$ MeV（对于 $M \sim 10^{15}$ g PBH）
    \item 信号：与标准霍金谱相比增强的伽马射线尾
    \item 幅度：$E \sim 10 \times T_H$ 处增强2--5倍
\end{itemize}

\textbf{当前约束}：Fermi-LAT CGB限制 $f_{PBH} < 10^{-8}$

\textbf{CTA改进}：对 $f_{PBH} \sim 10^{-10}$ 的灵敏度（100倍提升）

\textbf{证伪判据}：如果CTA未观测到超出或谱形与CTUFT预测不符

\subsubsection{预测2：PBH质量谱峰}
\label{subsec:prediction-mass-zh}

\textbf{观测}：微引力透镜巡天（Subaru HSC、OGLE、下一代）

\textbf{预测}：
\begin{itemize}
    \item 特征质量：$M_{peak} \sim 10^{15}$ g
    \item 起源：通过蒸发时标与 $\tau_0 = 0.005$ 相联系
    \item 谱形：窄峰而非宽分布
\end{itemize}

\textbf{物理关联}：
\begin{equation}
t_{evap} = \frac{5120\pi G^2 M^3}{\hbar c^4} \approx 13.8 \text{ Gyr（对于 } M \sim 10^{15} \text{ g）}
\label{eq:evaporation-time-zh}
\end{equation}

\textbf{证伪判据}：如果微透镜巡天在 $M \sim 10^{15}$ g处未发现峰，或峰位与$\tau_0$导出的预测不符

\subsubsection{预测3：正电子谱修正}
\label{subsec:prediction-positron-zh}

\textbf{观测}：AMS-02、DAMPE（空间宇宙线探测器）

\textbf{预测}：
\begin{itemize}
    \item 能量范围：$E \gtrsim 10$ GeV
    \item 信号：来自PBH蒸发的过量正电子
    \item 谱形：被 $d_s \to 10$ 增强所修正
\end{itemize}

\textbf{当前状态}：AMS-02在 $E \gtrsim 10$ GeV观测到正电子超出（可能与暗物质或天体物理有关）

\textbf{CTUFT检验}：高能谱形可以区分PBH起源

\textbf{证伪判据}：如果高分辨率数据排除CTUFT谱形

\subsubsection{预测4：PBH并合率}
\label{subsec:prediction-merger-zh}

\textbf{观测}：LISA空间引力波探测器（2030s年代发射）

\textbf{预测}：
\begin{itemize}
    \item 质量范围：$10^{-12} - 10^{-7}$ M$_\odot$（LISA敏感）
    \item 并合率：依赖于PBH丰度 $f_{PBH}$
    \item 可探测性：暗物质的 $f_{PBH} \gtrsim 10^{-3}$ 部分
\end{itemize}

\textbf{LISA灵敏度}：特征应变 $h_c \sim 10^{-23}$ 在 $f \sim 1$ mHz

\textbf{证伪判据}：如果PBH构成暗物质的 $f \gtrsim 10^{-3}$ 但未探测到并合信号

\subsubsection{预测5：多信使一致性}
\label{subsec:prediction-multi-zh}

\textbf{观测}：伽马射线、中微子、引力波数据的联合分析

\textbf{预测}：
\begin{itemize}
    \item 相同PBH群体在所有波段产生信号
    \item 独立测量的PBH丰度 $f_{PBH}$ 一致
    \item 所有通道都有CTUFT修正谱
\end{itemize}

\textbf{关键检验}：以下之间的交叉关联：
\begin{itemize}
    \item CTA伽马射线超出
    \item IceCube-Gen2中微子事件
    \item LISA引力波暴
\end{itemize}

\textbf{证伪判据}：如果不同信使给出不一致的PBH性质

\subsection{当前观测约束}
\label{sec:current-constraints-zh}

\subsubsection{宇宙伽马射线背景（CGB）}

Fermi-LAT对各项同性伽马射线背景的测量约束了PBH丰度：

\begin{equation}
f_{PBH} \lesssim 10^{-8} \quad \text{对于} \quad M \sim 10^{15} \text{ g}
\label{eq:cgb-constraint-zh}
\end{equation}

\textbf{CTUFT含义}：如果此质量尺度的PBH存在，其辐射必须与CTUFT谱一致。

\subsubsection{微引力透镜约束}

当前微引力透镜巡天的限制：

\begin{itemize}
    \item \textbf{Subaru HSC}：$f_{PBH} < 0.01$ 对于 $10^{11} - 10^{17}$ g
    \item \textbf{EROS}：$f_{PBH} < 0.1$ 对于 $2\times10^{-7} - 10^{-4}$ M$_\odot$
    \item \textbf{OGLE}：$f_{PBH} < 0.1$ 对于 $10^{-6} - 10^{-3}$ M$_\odot$
\end{itemize}

\subsubsection{正电子约束}

AMS-02对正电子分数的测量在 $E \gtrsim 10$ GeV显示超出。虽然通常归因于脉冲星或暗物质，CTUFT提供了可通过谱形检验的可测试替代解释。

\subsection{结论}
\label{sec:exp-conclusion-zh}

CTUFT做出了五项具体的、实验可检验的预言，将其与标准霍金辐射和其他量子引力方法区分开来：

\begin{enumerate}
    \item \textbf{伽马射线增强}：CTA可探测（2025+）
    \item \textbf{质量谱峰}：微引力透镜可检验
    \item \textbf{正电子超出}：可区分的谱形
    \item \textbf{引力波}：LISA可探测的PBH并合
    \item \textbf{多信使一致性}：跨波段关联
\end{enumerate}

理论的单参数$\tau_0 = 0.005$决定了可观测PBH的特征质量尺度，在基础物理与观测天文学之间建立了直接联系。

\textbf{关键信息}：CTUFT不仅仅是一个理论框架，而是一个将在未来10--15年内被观测验证或证伪的\textbf{物理可检验理论}。
