# TNN小鼠实验设计

## 概述

基于TNN数字果蝇实验（560K参数）的成功，设计更大规模的**TNN小鼠实验**。小鼠作为哺乳动物模型，具有更复杂的行为和神经系统，是验证TNN架构可扩展性的理想下一步。

---

## 1. 实验目标

### 1.1 科学目标
- 验证TNN架构在哺乳动物级别的可扩展性
- 实现小鼠核心行为：探索、社交、恐惧、学习
- 与真实小鼠行为数据对比验证
- 测试长期记忆和情境学习

### 1.2 技术指标
| 指标 | 目标 | 对比 |
|------|------|------|
| 参数量 | 10-50M | 果蝇实验560K的20-90倍 |
| 实时因子 | >0.5x | 略慢于实时可接受 |
| 行为数量 | 8+ 种 | 果蝇5种的扩展 |
| 记忆时长 | 分钟级 | 果蝇秒级的扩展 |

## 2. 架构设计

### 2.1 神经网络架构

```
TNN小鼠大脑架构
================

输入层 (感觉系统)
├── 视觉系统: 128×128 RGB = 49,152输入
├── 听觉系统: 频率×时间 = 1,024输入  
├── 嗅觉系统: 400种受体 = 400输入
├── 触觉系统: 身体表面 = 256输入
├── 本体感觉: 关节角度 = 64输入
└── 内部状态: 饥饿/恐惧等 = 32输入
    总计: ~51K 感觉输入

互反空间 (处理流)
├── 初级感觉皮层: 4层 × 1024维
├── 联想皮层: 6层 × 2048维  
├── 运动皮层: 4层 × 1024维
└── 海马体: 3层 × 512维 (空间记忆)
    总计: ~25K 互反空间维度

内部空间 (高级表征)
├── 高级视觉: 2层 × 4096维
├── 多模态整合: 3层 × 8192维
├── 决策中心: 2层 × 4096维
└── 工作记忆: 1层 × 2048维
    总计: ~45K 内部空间维度

输出层 (运动系统)
├── 前肢运动: 32×2 = 64输出
├── 后肢运动: 32×2 = 64输出
├── 头部控制: 16输出
├── 尾巴控制: 8输出
└── 行为选择: 16输出
    总计: ~168 运动输出

预估总参数量: 25-35M
```

### 2.2 关键创新

**1. 海马体空间细胞模拟**
- 位置细胞 (Place Cells): 编码空间位置
- 网格细胞 (Grid Cells): 空间导航
- 头方向细胞 (Head Direction): 朝向感知

**2. 社交认知模块**
- 同伴识别网络
- 社交等级学习
- 亲子依恋模拟

**3. 情绪状态机**
- 探索/利用权衡
- 恐惧/安全切换
- 奖赏/惩罚学习

## 3. 感官系统设计

### 3.1 视觉系统
```python
视觉处理流程:
视网膜 (128×128 RGB)
    ↓
初级视觉皮层V1 (边缘检测、方向选择)
    ↓
次级视觉皮层V2 (纹理、颜色)
    ↓
高级视觉皮层 (物体识别、运动检测)
    ↓
空间认知 (距离估计、路径规划)

特性:
- 视野: 140°水平 × 120°垂直
- 深度感知: 立体视觉模拟
- 运动敏感: 光流计算
```

### 3.2 听觉系统
```python
听觉处理:
音频输入 (16kHz采样)
    ↓
频谱分析 (64个频带)
    ↓
时频特征提取
    ↓
声音识别 (同类叫声、危险信号)
    ↓
声源定位 (双耳时间差)

关键声音:
- 同类超声波交流 (30-100kHz)
- 危险警报声
- 环境声音 (脚步声等)
```

### 3.3 嗅觉系统
```python
嗅觉通路:
400种化学受体
    ↓
嗅球处理 (模式分离)
    ↓
梨状皮层 (气味识别)
    ↓
杏仁核 (情绪关联)
    ↓
海马体 (空间关联)

重要气味:
- 同类气味识别
- 捕食者气味
- 食物气味
- 领地标记
```

### 3.4 触觉系统
```python
触觉分布:
- 胡须: 64传感器 (主要探索工具)
- 身体表面: 128传感器
- 爪垫: 64传感器
- 尾巴: 32传感器

功能:
- 纹理识别
- 振动检测
- 空间映射 (胡须扫动)
- 疼痛/不适感知
```

## 4. 运动系统设计

### 4.1 身体模型
```
小鼠身体结构:
==============

躯干 (椭球体)
├── 质心位置: (x, y, z)
├── 朝向: (yaw, pitch, roll)
└── 尺寸: 8cm × 3cm × 3cm

头部
├── 颈部关节: 3DOF
├── 眼睛: 2个 (立体视觉)
├── 耳朵: 2个 (可动)
└── 胡须: 双侧各32根

四肢
├── 前肢: 肩关节(3DOF) + 肘(1DOF) + 腕(2DOF)
├── 后肢: 髋关节(3DOF) + 膝(1DOF) + 踝(2DOF)
└── 爪子: 可抓取/挖掘

尾巴
├── 节段: 10节
└── 功能: 平衡、温度调节、情感表达
```

### 4.2 运动控制
```python
步态生成器:
├── 慢速行走 (Walk): 对角线步态
├── 快速行走 (Trot): 对角同步
├── 奔跑 (Gallop): 前-后交替
├── 转身 (Turn): 差动转向
└── 精细操作: 前爪协调

运动基元:
├── 前进/后退
├── 左转/右转
├── 探索性嗅探
├── 直立观察
├── 挖掘动作
└── 梳理动作
```

## 5. 行为模块设计

### 5.1 核心行为

**1. 探索行为 (Exploration)**
```python
触发条件: 好奇心 > 阈值，安全环境
子行为:
- 开放场地探索 (Open Field)
- 新物体调查 (Novel Object Recognition)
- 路径规划与空间学习

评估指标:
- 探索时间比例
- 中心区域停留时间 (焦虑指标)
- 路径效率
```

**2. 社交行为 (Social Behavior)**
```python
触发条件: 检测到同类气味/声音
子行为:
- 同伴接近与跟随
- 嗅觉调查 (Sniffing)
- 社交等级确立
- 亲子互动 (繁殖期)

评估指标:
- 社交互动频率
- 跟随行为持续时间
- 社交偏好测试
```

**3. 恐惧/防御行为 (Fear/Defense)**
```python
触发条件: 危险信号 (捕食者气味、巨响)
子行为:
- 冻结反应 (Freezing)
- 快速逃跑 (Escape)
- 防御性攻击
- 躲藏行为

评估指标:
- 冻结持续时间
- 逃跑路径选择
- 恐惧习得速度
```

**4. 觅食行为 (Foraging)**
```python
触发条件: 饥饿信号，食物气味检测
子行为:
- 食物搜索
- 取食动作
- 食物囤积 (Scatter Hoarding)
- 消费-返回权衡

评估指标:
- 觅食效率
- 路径记忆准确性
- 囤积策略
```

**5. 筑巢行为 (Nesting)**
```python
触发条件: 温度低或准备繁殖
子行为:
- 材料收集
- 巢穴位置选择
- 筑巢动作协调

评估指标:
- 巢穴质量评分
- 材料使用效率
```

**6. 理毛行为 (Grooming)**
```python
触发条件: 卫生需求或压力缓解
子行为:
- 面部理毛
- 身体理毛
- 尾巴理毛

评估指标:
- 理毛频率和时长
- 序列完整性
```

**7. 睡眠/休息 (Sleep/Rest)**
```python
触发条件: 疲劳、黑暗、安全
子行为:
- 位置选择
- 姿势调整
- 睡眠阶段 (浅/深)

评估指标:
- 睡眠周期
- 位置偏好
```

**8. 学习行为 (Learning)**
```python
任务类型:
- 迷宫学习 (Morris Water Maze)
- 条件反射 (Fear Conditioning)
- 工具使用学习
- 社交学习

评估指标:
- 学习曲线
- 记忆保持时间
- 泛化能力
```

## 6. 学习与记忆系统

### 6.1 记忆类型

**1. 工作记忆 (秒级)**
- 位置保持
- 任务规则
- TNN内部空间实现

**2. 短期记忆 (分钟级)**
- 最近事件
- 社交互动历史
- 海马体实现

**3. 长期记忆 (小时-天)**
- 空间地图
- 恐惧记忆
- 技能学习
- 突触可塑性实现

### 6.2 学习机制

**Hebbian可塑性**
```python
# TNN扭转场可塑性
Δτ_ij = η · pre_i · post_j · f(timing)

其中:
- η: 学习率
- pre_i: 前突触活动
- post_j: 后突触活动
- f(timing): 时间窗口函数
```

**强化学习**
- 奖赏预测误差 (RPE)
- 多巴胺信号模拟
- 策略梯度更新

## 7. 验证实验设计

### 7.1 标准行为测试

**1. 开放场地测试 (Open Field Test)**
```
设置: 50cm × 50cm 方形开放区域
测量:
- 总移动距离
- 中心区域停留时间 (焦虑指标)
- 边缘偏好
- 垂直探索 (rearing)

预期行为: 
- 初期边缘徘徊
- 逐渐进入中心区域
- 探索与回避权衡
```

**2. 新物体识别 (Novel Object Recognition)**
```
阶段1: 熟悉期 - 探索两个相同物体 (5分钟)
阶段2: 间隔期 - 移出小鼠 (1小时)
阶段3: 测试期 - 一个熟悉物体 + 一个新物体

测量: 对新物体的探索时间比例
预期: >50%时间探索新物体 (记忆识别)
```

**3. 社交偏好测试 (Three-Chamber Test)**
```
设置: 三个连通室
- 左侧: 陌生小鼠A
- 中间: 空
- 右侧: 新物体

测量:
- 在陌生小鼠侧的时间
- 在社交互动的时间

预期: 偏好陌生小鼠 > 新物体
```

**4. 恐惧条件反射 (Fear Conditioning)**
```
训练: 声音CS + 电击US (3次配对)
测试1: 24小时后，声音 alone
测试2: 情境依赖 (新环境 vs 训练环境)

测量: 冻结反应百分比
预期: 条件刺激引发冻结
```

**5. Morris水迷宫 (Morris Water Maze)**
```
设置: 1.5m直径圆形水池，隐藏平台

训练阶段 (5天):
- 每天4次试验，从不同起点
- 记录找到平台的潜伏期

探测试验 (第6天):
- 移除平台
- 记录目标象限停留时间

预期: 学习后快速找到平台，偏好目标象限
```

**6. 社交等级测试 (Tube Test)**
```
设置: 狭窄透明管道

测试:
- 两只小鼠从两端进入
- 记录哪只后退

重复: 10次，确定主导/从属关系
```
