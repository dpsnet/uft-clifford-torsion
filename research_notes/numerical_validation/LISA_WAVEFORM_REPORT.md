## 3. 关键物理特征

### 3.1 6偏振模式的物理来源

**标准GR (2种)**:
- plus/cross: 横向无迹张量模式，传播速度 = c

**UFT额外 (4种)**:
- vector_x/vector_y: 扭转场的矢量激发，振幅 ~ τ₀
- breathing: 标量呼吸模式，振幅 ~ τ₀²
- longitudinal: 纵向模式，振幅 ~ τ₀²

### 3.2 传播速度修正

不同偏振模式的引力波传播速度:

```
v_plus = v_cross = c
v_vector = c × (1 - 0.1 τ₀²)
v_breathing = c × (1 - 0.2 τ₀²)  
v_longitudinal = c × (1 - 0.3 τ₀²)
```

对于τ₀ = 10⁻⁶:
- 速度修正 ~ 10⁻¹⁰
- 长距离传播产生可探测的相位差

### 3.3 振幅修正

UFT修正因子:
- 矢量偏振: A_vector = 0.5 τ₀ × A_plus ~ 5×10⁻⁶
- 标量偏振: A_scalar = 0.3 τ₀² × A_plus ~ 3×10⁻¹¹

**探测策略**:
- 额外偏振能量占比 ~ τ₀² ~ 10⁻¹⁰
- 需要网络探测器的偏振分析
- LISA三角构型可部分区分偏振模式
