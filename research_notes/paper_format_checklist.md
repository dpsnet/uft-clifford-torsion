# 论文格式统一检查清单

## 字体与排版
- [ ] 正文字体: 中文宋体10.5pt, 英文Times New Roman 10.5pt
- [ ] 标题字体: 黑体(中文), Arial(英文)
- [ ] 公式编号: (章节.序号), 如 (3.2)
- [ ] 图表编号: 图1, 图2, ... 或 Fig.1, Fig.2 (全文统一)
- [ ] 参考文献: APA格式或目标期刊指定格式

## 图表规范
- [ ] 分辨率: 300 DPI (印刷)
- [ ] 格式: PDF矢量图优先
- [ ] 字体: 图表内字体不小于8pt
- [ ] 配色: 科学标准配色 (蓝#0066CC, 红#CC0000, 绿#009900)

## 数学公式
- [ ] 公式环境: LaTeX equation环境
- [ ] 符号统一: 全文统一符号定义
- [ ] 定理环境: 定理/引理/证明环境统一

## 章节结构
```
摘要 (中英文)
1. 引言
2. 数学基础
   2.1 ... 2.2 ... 2.3 ... 2.4 ...
3. 理论框架
   ...
(中间章节)
...
11. 结论

附录A: 定理证明
附录B: 代码与数据
附录C: 严格论证

参考文献
```

## 语言规范
- [ ] 中文: 科技论文规范用语
- [ ] 英文: 学术英语, 避免口语化
- [ ] 术语统一: 全文术语一致性检查

## 参考文献格式
- [ ] 期刊: 作者. 标题. 期刊名, 卷(期): 页码, 年份.
- [ ] 书籍: 作者. 书名. 出版社, 年份.
- [ ] arXiv: 作者. 标题. arXiv:编号, 年份.

## 提交前检查
- [ ] 字数统计: 目标~20万字
- [ ] 页数统计: 目标~80页
- [ ] 图表数量: 13张核心图 + 4张数学附录图
- [ ] 参考文献: 100+
- [ ] 重复率: 使用查重工具检查
